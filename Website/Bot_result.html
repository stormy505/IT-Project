Как выгрузить бота на сервер?



А чтобы бот работал постоянно, можно воспользоваться онлайн хостингом от pythonanywhere. Для этого нужно зайти на сайт по ссылке, зарегистрироваться и создать новую директорию с файлом, где будет храниться код с ботом. Далее запустить терминал и установить через него уже знакомую нам библиотеку:

pip install pyTelegramBotAPI

Переходим на вкладку с нашим кодом и нажимаем кнопку Run. Готово! Теперь у нас есть свой бот-помощник!

Исходный код можно найти на веб-сервисе GitHub (веб-сервис для хостинга IT-проектов и их совместной разработки).

Также в выгруженном репозитории можно будет найти папку с исходным кодом сайта.
